# Documentación - Mundo Reporte

Este directorio contiene toda la documentación del proyecto Mundo Reporte.

## Archivos

- **RESUMEN_PROYECTO.md** - Descripción general del proyecto y características
- **INSTRUCCIONES_INICIO.md** - Guía para iniciar el desarrollo local
- **SUPABASE_SETUP.md** - Configuración de Supabase para la base de datos
- **SUPABASE_VERIFICACION.md** - Cómo verificar la conexión a Supabase
- **RENDER_CONFIG.md** - Configuración para desplegar en Render
- **SOLUCION_DESPLIEGUE.md** - Soluciones para problemas de despliegue
- **LIMPIAR_CACHE.md** - Cómo limpiar caché de dependencias
- **ACCION_INMEDIATA.md** - Acciones urgentes o pendientes
- **README.md** - (en raíz) Documentación principal del proyecto

## Inicio Rápido

1. Leer **RESUMEN_PROYECTO.md** para entender el proyecto
2. Seguir **INSTRUCCIONES_INICIO.md** para configurar el ambiente local
3. Consultar **SUPABASE_SETUP.md** si necesitas configurar la base de datos
